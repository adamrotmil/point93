Save & Resume: Auto-save after each question. Browser close → return days later → resumes at next unanswered question. No timeout in v1.

📋 Content System Rules Applied to Results (§9)
  • Three-Voice System: V1 Point93 Analytical (Gray/Teal) · V2 Client Composite (Gold) · V3 External Benchmark (Green)
  • Five Treatment Types AI Generated: Key Finding · In Practice · Client Voice · Top Performers · Your Action Plan
  • Gap Priority drives treatment count: GP1=3 treatments, GP2=2, GP3=2, GP4+=1
  • Client Voice scarcity: Max 2 per module (1 Overview + 1 GP1 IA). Non-negotiable.
  • Band-Adaptive language: tone shifts by band (Top Tier, Competitive, Needs Work, At Risk)
Flow 3: BI Access — Dual Access Model
Blueprint §7.1–7.6
Business Intelligence products use a Dual Access Model: advisors can earn access through module completion prerequisites, or purchase individual BI profiles as standalone products. BI is NOT tier-gated. Stage-based progression is independent from the module (BARS) assessment track.
BI Overview Page
§7.1
Shows all 7 registered BI products: PP, CR, COI, RT, DQ, HH, NG.
  • Each product card shows: name, stage count, access status
  • BI branches from P93 root (not from any area/module)
  • Visually distinct from module assessment path
Click BI product card
BI Product Home
§7.3
Gateway for each BI profile.
  • Stage Map (hero component): sequential, unlockable stages
  • Stage states: Locked (muted, lock icon, prerequisite visible) · Available (highlighted, actionable) · Completed (check, results accessible)
  • Cumulative intelligence: completed stages remain expanded
How to unlock?
Dual Access Decision
§7.2
Two parallel paths to BI access
PATH 1 — EARN
Complete Prerequisite Modules
§7.2 Path 1
Complete prerequisite modules to unlock BI stages progressively.
  • Each BI stage has specific module prerequisites
  • Stages unlock sequentially as prerequisites are met
  • Partial data produces partial intelligence
  • More stages = richer results
Stage Unlocked → Take BI Questions
Each BI profile has its own questions, stages, and scoring independent from module assessments. Stage transitions should feel meaningful and rewarding.
PATH 2 — PURCHASE
Purchase BI Profile
§7.2 Path 2
Buy individual BI profiles as standalone products.
  • Individual: $25 per BI profile
  • Bundle: $125 for all 7 profiles
  • Bypasses module prerequisites
  • All stages available immediately
Full Access → Take All BI Questions
All stages unlocked. Complete questions per stage to build cumulative intelligence profile.
Both paths converge → BI Results
BI Results (Cumulative)
§7.3–7.5
Stage results build cumulatively. Completed stages remain expanded.
  • BI scores do NOT feed into module scores or overall P93 score
  • Results appear only on BI pages
  • Exception: Moenio Nudge can blend BI results into recommendations
  • Self-contained architecture: branches from P93 root
🎨 BI Visual Differentiation Mandate (§7.4)
  • Distinct visual identity: differentiated color palette, layout pattern, or visual treatment from module assessment path
  • Stage progression emphasis: Stage Map is hero component with clear locked/available/completed states
  • NO tier-gating language or patterns: no tier upgrade CTAs, no tier badges on BI pages
  • Cumulative intelligence display: completed stages remain expanded, building a growing intelligence profile
Critical Rule: BI ≠ Tier-Gated
BI access is governed solely by module completion prerequisites (earn) or optional purchase ($25/$125). BI pages must NOT display tier upgrade CTAs, tier badges, or any visual element implying content is restricted by subscription tier. This is a design mandate (§7.4).

Edge Cases — Retake (§5.9)
Save & Resume during retake: Same behavior as first-time — auto-save after each question, resume at next unanswered.

Band changes: If retake moves advisor between bands, all band-adaptive language regenerates across Module Results.
