Four-Band System — §11.1
Bands are dynamic — thresholds shift as dataset grows. Applied to every scored entity: data point, IA, module, area, overall. Band Formula: CF = max(A, T−K), VF = min(CF−K, A), K=15.

Band Formula — §11.1
CF = max(A, T−K) · VF = min(CF−K, A) · K = 15
Example: T=90, A=70 → CF=75, VF=60. Bands: 75+ Top Tier, 60–74 Competitive, below 60 At Risk.

Blueprint §3.7, §5.9 — Empty states, locked states, coming soon
Every screen has defined states for pre-assessment, partial completion, full completion, and tier-locked content. No blank sections ever appear.
Empty State Matrix

Page structure shown with guidance text. CTA to begin first assessment. Shows what will appear. No blank sections.
Partial Completion
Available data displayed. Remaining sections show: "Complete [Module X] to see [Component Y]." No blank sections.
Full Completion

Tier-Locked
Scores always visible. Intelligence depth varies. One upgrade prompt per section, not per item. Never blocks access to scores.

Blueprint §5.5, §3.6 — Three-Act Reveal, Cascade Reveal
Two primary animation systems: the Three-Act Reveal (module-level, first visit) and the Cascade Reveal (dashboard-level, progressive unlock). Both are one-time animations, not gating mechanisms.
Three-Act Reveal — §5.5
Sequential reveal on Module Results page. First visit after completion only. One-time animation — not a gating mechanism.

AI Denotation System
Applied across all deliverables — Denotes AI-generated narrative content
All narrative content on the Point93 platform is AI-generated. This denotation system provides three levels of visual marking to indicate AI-produced content, from prominent badges to subtle inline markers.
Denotation Levels
Level 1 — AI Badge (Prominent)
AI GENERATED Used on: Section headers, treatment type labels, voice system titles
Level 2 — AI Marker (Compact)
AI Generated Used on: Inline references, sub-component labels, flow node annotations
Level 3 — AI Border (Structural)
Content block with gradient left border (blue → teal). Used on: Treatment block containers, voice sample previews, narrative output blocks.

Rule — Universal AI Denotation
All narrative content on the Point93 platform is AI-generated. The denotation system ensures transparency for advisors. Scores and raw data points are NOT AI-generated — only the interpretive narrative layer carries the AI denotation.
